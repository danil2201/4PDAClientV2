# 4PDAClient
My attempt to recover discontinued [4PDAClient app](https://www.microsoft.com/ru-ru/p/4pdaclient/9nblggh0ggvj).

## About
Simple UWP application to display 4PDA web site on "winphone" with Windows 10 Mobile.
